# 汉字笔顺练习

一个帮助学生练习汉字笔顺的 Web 应用。

## 功能特点

- 支持四、五、六年级的汉字练习
- 可以添加和删除汉字
- 支持单个或批量添加汉字
- 笔顺动画演示
- 练习模式

## 安装和运行

1. 克隆仓库：
bash
git clone https://github.com/wuqiangxjtu/bishun.git

2. 安装依赖：
```bash
pip install -r requirements.txt
```

3. 运行应用：
```bash
python app.py
```

## 使用说明

1. 选择年级
2. 添加需要练习的汉字
3. 点击汉字进行练习或使用"全部练习"功能
